Variables, Types, Operators
===============================

Variables
----------

.. code-block:: javascript

    var count = 5;
    let message = "Hello World";
    const likesJS = true;

JavaScript variables are *loosely* typed meaning we can change their type at will.


Operators
----------


.. code-block:: javascript
    
    var product = 3.14159 + 2.71828; // 5.85987
